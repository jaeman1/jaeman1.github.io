## **Nmap 스캔 옵션**

- `-sT` : `connect()` 함수를 이용한 Open 스캔
- `-sS` : 세션을 성립시키지 않는 SYN 스캔
- `-sF` : FIN 패킷을 이용한 스캔
- `-sN` : NULL 패킷을 이용한 스캔
- `-sX` : XMAS 패킷을 이용한 스캔
- `-sP` : Ping을 이용한 호스트 활성 여부 확인
- `-sU` : UDP 포트 스캔
- `-b` : FTP 바운스 스캔

## **Nmap 실행 옵션**

- `-f` : 방화벽을 통과할 수 있도록 패킷을 설정
- `-v` : 스캔의 세부 사항을 출력
- `-PI` : 시스템 활성화 여부를 ICMP로 검사
- `-O` : 시스템의 운영체제를 추정
